About this manual

This document describes the file structure of TYPO3 and takes you step by step through the installation. It does not focus on specific operating systems, but contains some hints on general TYPO3 installation issues with regards to Apache, PHP and MySQL.

This document is included as part of the official TYPO3 documentation.
